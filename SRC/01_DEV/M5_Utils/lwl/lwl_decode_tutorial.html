📘 LWL Log Decode Tool Tutorial

This guide will help you use the lwl_decode.py script to decode binary log files generated by the LWL (Lightweight Logging) system in embedded projects.

🧠 Purpose

lwl_decode.py converts a binary .bin log file into a readable .txt file using metadata extracted from the lwl_msg_table[] defined in C source files (e.g., lwl.c).

🗂️ Required Files

Make sure your project includes:

project_folder/
│
├── lwl.c               # Contains the lwl_msg_table[] definition
├── lwl.h               # Header file for LWL logging (optional for decode)
├── lwl_decode.py       # The decoder script
├── log.bin             # Your binary log file (can be renamed)
└── output.txt          # The decoded human-readable log output

🚀 How to Use

🔧 Basic Usage

python lwl_decode.py
  • Uses: ./lwl.c as default source file
  • Input: exp_log_20250805_082000.bin
  • Output: output.txt

📂 Use Custom Source File

python lwl_decode.py -f path/to/source.c

Use your own .c file containing lwl_msg_table[].

📁 Parse Entire Directory

python lwl_decode.py -d path/to/source_folder/

Search for lwl_msg_table[] in all .c files under the specified folder.

🛠 Specify Input and Output

python lwl_decode.py --bin log_01.bin --output decoded_log.txt

Specify binary log input and text output.

🐞 Enable Debug Log

python lwl_decode.py --log debug

Prints detailed debug info during decoding.

🧩 Log Format Details

Binary log format:

[0xAA][LENGTH][ID][ARG BYTES][CRC]

Where:

  • 0xAA: Start byte
  • LENGTH: Number of bytes = ID + ARGS + CRC
  • ID: Message ID defined in lwl_msg_table[]
  • ARG BYTES: Data arguments (little-endian)
  • CRC: CRC-8-ATM calculated over [ID + ARGS]

🛠 Supported Format Strings

In lwl_msg_table[], you can use:

  • %1d → 1 byte integer
  • %2d → 2 byte integer
  • %4d → 4 byte integer
  • %% → Literal percent symbol

Example definition:


static const struct lwl_msg lwl_msg_table[] = {
    {"Timestamp %4d", 4},     // ID 0
    {"Temperature %2d", 2},   // ID 1
    {NULL, 0},                // ID 2
};

✅ Output Example

Given binary log:

AA 06 02 06 1B 01 96

With format "Temperature: NTC[%1d]: %2d", output will be:

Temperature: NTC[6]: 283
